# seccount - toy series
[README in Japanese](README.ja.md)

シンプルな秒数カウンターウェブアプリケーション。

## デモ
https://code4fukui.github.io/seccount/

## 機能
- ページが読み込まれてから経過した秒数を表示します
- 画面をクリックすることでカウンターの開始/停止ができます
- 数字の桁数に合わせてフォントサイズが自動的に調整されます

## 使用方法
1. [デモページ](https://code4fukui.github.io/seccount/)をウェブブラウザで開いてください。
2. 画面をクリックすると、カウンターを開始/停止することができます。
